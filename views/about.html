About The Site

This web development project involves building a dynamic website using Node.js as its backend framework, which facilitates efficient, event-driven server-side operations. The website exposes a RESTful API that enables standardized communication between the client and server using JSON as the data interchange format. Client-side functionality is managed with JavaScript, creating an interactive and responsive user experience, while MongoDB serves as the flexible, document-oriented database. This setup not only supports rapid development and scalability but also ensures that the system can handle real-world applications with robust performance and seamless integration of its components.

This website is designed to serve as a secure messaging platform. It accepts a user’s message, then applies either RSA encryption, which is reversible under controlled conditions, or an irreversible one-way encryption function to ensure the confidentiality of the content. Once encrypted, the message is posted, maintaining both data integrity and privacy throughout the process.
