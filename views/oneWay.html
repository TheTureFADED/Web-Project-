Basic One-Way Hash Function

A one-way hash function converts input data of any size into a fixed-size output, where it's practically impossible to reconstruct the original input from the output. This simple implementation demonstrates the basic principles.

How This Basic Hash Function Works

This is a simple hash function that demonstrates the core concepts of hashing without cryptographic security. Here's how it works:

  1. Each character in the input is converted to its ASCII code
  2. The ASCII codes are combined using a simple mathematical formula
  3. The result is converted to a hexadecimal string of fixed length
// Basic One-Way Hash Function function basicHash(input) { let hash = 0; // If the input is empty, return a default hash if (input.length === 0) return "00000000"; // Process each character in the input for (let i = 0; i < input.length; i++) { // Get the character code const char = input.charCodeAt(i); // Combine it with the current hash value // (multiply by 31 and add character code) hash = ((hash << 5) - hash) + char; // Convert to 32-bit integer hash = hash & hash; } // Convert to hexadecimal and ensure 8 characters let hexHash = (hash >>> 0).toString(16); // Pad with zeros if necessary while (hexHash.length < 8) { hexHash = "0" + hexHash; } return hexHash; }
Key Properties of a One-Way Hash Function
  • Fixed Output Length: This hash function always produces an 8-character hexadecimal output, regardless of input size.
  • Deterministic: The same input will always produce the same output.
  • One-Way: It should be difficult to reverse the hash to find the original input.
  • Avalanche Effect: A small change in the input produces a significantly different output.
Note: This is a simplified hash function for educational purposes only. It is not cryptographically secure and should not be used for password storage, data verification, or other security-critical applications. For real security applications, use established algorithms like SHA-256 or SHA-3.
